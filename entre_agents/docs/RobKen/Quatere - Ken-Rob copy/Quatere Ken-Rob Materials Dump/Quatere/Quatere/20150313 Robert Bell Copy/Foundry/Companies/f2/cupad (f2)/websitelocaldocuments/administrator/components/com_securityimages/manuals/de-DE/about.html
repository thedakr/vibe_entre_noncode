SecurityImages mit CAPTCHA für Joomla! 1.5

Security images (Sicherheitsbilder)sind dynamisch erzeugte Bilder, welche einen für Maschinen schwer lesbaren Text enthalten. Es wird eine Kombination aus verschiedenen Schriftzeichen, Farben, Schriftwinkeln und Hintergründen benutzt um Computern das automatische auslesen zu erschweren und so SPAM zu verhindern. Der (menschliche) Benutzer hat alle Zeichen korrekt einzugeben bevor eine Eintragung möglich wird.
