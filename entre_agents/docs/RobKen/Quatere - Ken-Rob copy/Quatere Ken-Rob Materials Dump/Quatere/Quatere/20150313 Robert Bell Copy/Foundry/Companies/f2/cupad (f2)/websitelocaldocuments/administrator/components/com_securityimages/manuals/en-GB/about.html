SecurityImages or CAPTCHA for Joomla! 1.5

Captcha are dynamically generated images containing hard to read text, it is using a combination of font colors, font size, fuzzy background, font angle in order to disallow computer to automate reading and also limit spamming.

The user (a human) has to reproduce all character correctly before being allowed to post.
